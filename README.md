## Features

- Real-time style switching between:
  - Modern
  - Neumorphic
  - Glassmorphic
  - Material
  - Gaming

- Customizable components:
  - Styled inputs
  - Styled buttons
  - Dynamic color schemes
  - Custom typography

## Building

1. Ensure Qt 6.2+ is installed
2. Clone the repository
3. Build using CMake or QtCreator:

```bash
mkdir build && cd build
cmake ..
make
```
